# 💰 FinanceManager

A simple Java application for managing personal finances, tracking income and expenses, and monitoring your financial balance.

---

## ℹ️ Overview

**FinanceManager** is an application that allows users to record transactions, organize them into categories, and view financial statistics. All data is stored locally in a CSV file, making the application easy to use.

- Java 23

## ✨ Features

- Add income and expense transactions
- Categorize financial records
- Calculate current balance
- Filter transactions by category
- View financial statistics
- Store data locally in CSV format

## ⬇️ Installation

### Requirements

- Java 23
- IntelliJ IDEA

### Clone Repository

```bash
git clone https://github.com/filipbures-1/FinanceManager.git
```

### Run Application

Open the project in IntelliJ IDEA and run:

```text
src/Main.java
```
